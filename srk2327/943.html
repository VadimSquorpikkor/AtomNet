Адаптер интерфейсный АИ-АТ943

Работа с SARK2_PXR
Настройка АИ-АТ943
Настройка с САТ32
943-й адаптер интерфейсный. Старая версия

Данный адаптер по функционалу является аналогом MOXA NE-5130 и служит для связи СРК с ПК посредством Ethernet

  • Получить модуль Ethernet MOXA NE-4120A
  • Вставить модуль в адаптер
    • Модуль имеет 3 группы контактов: две по два ряда (1 и 2) и одну однорядовую (3). Модуль вставлять так, чтобы двухрядовые (1 и 2) группы (вилки) полностью зашли в аналогичные разъёмы на плате адаптера. Соответственно однорядовый (3) зайдёт только в один (из двух) рядов соответствующего разъёма
  • Подключить питание к адаптеру (3) и кабель Ethernet (Cat.5) (4).
  • Подключить кабель Ethernet в ПК и запустить ПО "Network Enabler Administrator".
  • Настроить адаптер для работы с СРК при помощи инструкции.

    • Рабочее подключение адаптера в СРК
