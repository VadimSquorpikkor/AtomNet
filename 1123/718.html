ДКС-АТ1123. Проверка коэффициентов

  • При включении нажать и удерживать кнопку МЕМ
  • Нажать три раза МЕМ, на экране появится "---":
  • Два раза нажать кнопкуSTART
  • Набрать "718", для этого с помощью клавиш  △  и  ▽  выбирается цифра, а спомощью START выбирается следующая позиция
  • После ввода последней цифры, нажать МЕМ
  • Нажать и удерживать кнопку МЕМ, появится ".1."
  • Нажимая кнопку МЕМ, выбрать пятый режим
  • Переключение коэффициента осуществляется кнопкой START, значение коэффициента изменяется кнопками  △  и  ▽ 

Должны стоять такие коэффициенты:

1U 1.0
2U 1.0
3U 1.0
4U 0
5U 0
