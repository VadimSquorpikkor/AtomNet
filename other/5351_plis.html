Прошивка ПЛИС в ДКС-АТ5351

  • 1. Соединить программатор Lattice-USB-2A с ПК и ДКС-АТ5351 как на фото:
  • 2. Запустить программу Lattice Diamond.
  • 3. File → Open → Project → выбрать файл DKS_5351_MACHXO2_2000.ldf
  • 4. Tools → Programmer
  • 5. В поле "Device Family" выбрать "MachXO2", в поле "Device" выбрать LCMXO2-2000ZE, в поле "Operation" выбрать FLASH Erase, Program, Verify, в поле "File name" DKS_5351_MACHXO2_2000_impl1.jed
  • 6. Нажать "Program"
  • 7. По окончанию программирования в логе: Operation: successful.
