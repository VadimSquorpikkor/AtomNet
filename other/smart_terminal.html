Smart Terminal

Скачать последнюю версию (1.0.0.35)

Терминал под Android. Соединение — по Bluetooth.

Внешний вид. Что и как

  • 1 — поле отображения введенной команды
  • 2 — включить вибро (не работает)
  • 3 — Имя подключенного устройства
  • 4 — выключить вибро (не работает)
  • 5 — включить светлую тему (не работает)
  • 6 — включить темную тему (не работает)
  • 7 — открыть диалог поиска устройств Bluetooth
  • 8 — индикатор подключения. Если горит красным, устройство не подключено, если зеленым — подключено
  • 9 — закрыть программу (не работает)
  • 10 — индикатор неправильно введенной команды. Если ввести команду, состоящую из менее чем из 4-х символов или из нечетного количества, то такая команда не будет отправлена и загорится такой символ. При вводе или удалении символа индикатор исчезает
  • 11 — отправленная в устройство команда
  • 12 — полученный ответ
  • 13 — включение CRC (не работает). Включено по умолчанию
  • 14 — выбор типа CRC (не работает)
  • 15 — открыть проводник, чтобы открыть txt файл с командами. Если при выборе файла поставить флаг "Открывать автоматом", то эта кнопка сразу будет открывать этот выбранный файл в виде диалога со списком команд (без открывания проводника с поиском и выбором файла)
  • 16 — символы команды. При наборе команды не нужно вводить "пробел", после каждого второго символа "пробел" будет подставляться автоматом
  • 17 — отправка введенной команды
  • 18 — открыть список избранных команд
  • 19 — стереть последний символ
  • 20 — очистить поле введенной команды [1]
  • 21 — если включено, то к введенной команде будет добавлен префикс [22]
  • 22 — префикс

Файл с командами

Smart Terminal может работать с txt файлом команд обычного Терминала. Для этого нужно просто скопировать этот файл на телефон, затем в Smart Terminal нажать кнопку [15] и выбрать его в проводнике программы:

Файл будет загружен и открыт в виде диалога со списком всех команд и их описанием:

Если поставить флаг "Открывать автоматом", то при последующем нажатии кнопки [15] этот файл сразу откроется в диалоге без необходимости снова искать и выбирать файл в проводнике

Команды

При тапе по команде, текст команды копируется в поле введенной команды [1]

При долгом тапе по окне команды[1] или запросу[11] или ответу[12] появляется диалог с выбором доп. опций для выбранной команды:

Диалог состоит из следующих кнопок:

  • "Share" — при нажатии кнопки будет открыт системный диалог с выбором, куда и как передать текст выбранной команды (eMail, Viber, gDrive и т.д.). Это удобно, когда, например, нужно передать большой ответ от БД (спектр например)
  • "Ввод" — команда будет скопирована в поле введенной команды [1]
  • "Favorite" — команда будет добавлена в избранное (см. далее)
  • "Copy" — команда будет скопирована в буфер обмена

Избранное

Кроме возможности работать с файлом команд, есть вариант использовать "Избранное" для каких-то часто используемых команд. Для добавления команды в избранное нужно нажать кнопку "Favorite", после чего откроется диалог добавления команды в избранное. В диалоге можно отредактировать выбранную команду, а также добавить описание для неё

Открывается избранное по нажатии кнопки [18]
