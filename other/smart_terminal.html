Smart Terminal

Скачать последнюю версию (1.0.0.38)

Терминал под Android. Приложение позволяет посылать команды подключенному устройству, а также принимать и выводить ответ устройства на полученную команду. Подключение осуществляется через проводное соединение, с использованием подключенного адаптера интерфейсного БОИ-5 (только для Nautiz-X6 или Nautiz X81), либо посредством Bluetooth соединения.

Внешний вид. Что и как

  • 1 — список запросов/ответов;
  • 2 — отправленная команда. При тапе по команде, текст команды копируется в поле введенной команды [4]. Долгое нажатие открывает диалог управления выбранной командой (см. 4.4);
  • 3 — полученный ответ. Долгое нажатие открывает диалог управления полученного ответа;
  • 4 — поле отображения введенной команды;
  • 5 — выбор скорости подключения (доступно только при подключении по проводу);
  • 6 — имя подключенного устройства;
  • 7 — включить режим «на весь экран»;
  • 8 — переключение темы приложения (светлая или темная);
  • 9 — открыть диалог поиска устройств;
  • 10 — индикатор подключения. Если горит зелёным, устройство подключено, иначе горит красным;
  • 11 — очистить список [1];
  • 12 — открыть диалог управления введенной командой [4];
  • 13 — индикатор неправильно введенной команды. Если ввести команду, состоящую из менее чем из 2-х символов (плюс адрес) или из нечетного количества, то такая команда не будет отправлена и загорится такой символ. При вводе или удалении символа индикатор исчезает;
  • 14 — включение ввода CRC. При выключении CRC будет рассчитан автоматически и добавлен в конец команды;
  • 15 — ввод CRC (активен только при включенном флаге [14]);
  • 16 — открыть проводник, чтобы открыть txt файл с командами. Если при выборе файла поставить флаг "Открывать автоматом", то эта кнопка сразу будет открывать этот выбранный файл в виде диалога со списком команд (без открывания проводника с поиском и выбором файла);
  • 17 — открыть список «избранных» команд;
  • 18 — кнопки ввода команды. При наборе команды не нужно вводить "пробел", после каждого второго символа "пробел" будет подставляться автоматом;
  • 19 — отправить введенную команду;
  • 20 — открыть конвертер decimal → hex;
  • 21 — стереть последний символ;
  • 22 — очистить поле ввода команды;
  • 23 — если включен, к команде будет добавлен адрес [24];
  • 24 — адрес;

Подключение

Диалог поиска устройств запускается кнопкой [9]. После открытия диалога автоматически начнется поиск устройств для подкелючения по Bluetoth. Кнопка «Искать» включает поиск устройств, кнопка «Остановить» останавливает поиск. Для подключения к устройству через Bluetooth необходимо выбрать устройство из списка (найденных устройств либо ранее подключенных).

Для подключения к устройству подключенному по проводу (через БОИ-5) необходимо нажать кнопку «Подключиться к БОИ-5».

Файл с командами

Smart Terminal может работать с txt файлом команд обычного Терминала. Для этого нужно просто скопировать этот файл на телефон, затем в Smart Terminal нажать кнопку [15] и выбрать его в проводнике программы:

Файл будет загружен и открыт в виде диалога со списком всех команд и их описанием:

Если поставить флаг "Открывать автоматом", то при последующем нажатии кнопки [15] этот файл сразу откроется в диалоге без необходимости снова искать и выбирать файл в проводнике

Команды

При долгом тапе по окне команды[4] или запросу[2] или ответу[3] появляется диалог с выбором доп. опций для выбранной команды:

Диалог состоит из следующих кнопок:

  • Share — отправить выбранную команду через список доступных приложений Android (почта, Viber и т. д.);
  • Ввод — вставить полученную команду в поле ввода [4];
  • Favorite — добавить команду в список избранных команд
  • Copy — добавить команду в буфер обмена

Decimal/Hex конвертер

При необходимости ввести десятичное значение в 16-м виде можно использовать встроенный конвертер. С помощью цифровых кнопок вводится десятичное значение, в поле «HEX» сразу же будет отображаться значение в 16-м формате. При нажатии кнопки «ОК» полученное HEX значение будет добавлено в поле ввода команды

Избранное

Кроме возможности работать с файлом команд, есть вариант использовать "Избранное" для каких-то часто используемых команд. Для добавления команды в избранное нужно нажать кнопку "Favorite", после чего откроется диалог добавления команды в избранное. В диалоге можно отредактировать выбранную команду, а также добавить описание для неё
