Содержание / СРК / Блок регистрации

Блок регистрации. Прошивка.

  • Батарейка не нужна
  • Переставить перемычки (после прошивки вернуть, как было)
  • Программа "FlashMagic"
  • Подать питание 12V
  • Подсоединить кабель к X8
  • После перемычки вернуть перемычкуи, как было
  • Чтобы пульт работал с БР, пульт должен быть прошит SCR пршивкой
  • Выбрать тип блоека (БДКГ-25, скорее всего)

Настройка БР

  • К компу подключается через разъем "PU" к USB-COM адаптеру
  • Программа "SRK board"
    • Выбрать порт
    • Вкладка БДКГ-25
    • Соединить
  • Register device → Read
  • Connection & Device types → Read — покажет, какие типы блоков подключены (в этот момент они не обязаны быть подключенными) и какие адреса у блоков
  • При подключениик БР блоки получают адреся из этих ячеек
  • Settings → Read
